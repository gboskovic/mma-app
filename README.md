simple app that scrapes www.tapology.com and shows you upcoming matches of your favorite fighters  

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.8.

### Tech/framework used  

Built with

* Angular 8  
* Node.js  
* MySQL
